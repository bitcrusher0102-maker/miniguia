# 🧠 Miniguia de Estudo

## 🔐 Segurança Cibernética

É o conjunto de práticas utilizadas para proteger sistemas, redes e dados contra ataques digitais.

## ⚠️ Principais Ameaças

Malware: software malicioso
Ransomware: bloqueio de dados com pedido de resgate
Phishing: roubo de informações via engano
Man-in-the-Middle: interceptação de comunicação
DoS: sobrecarga de sistemas

## 🛡️ Boas Práticas

Manter sistemas atualizados
Utilizar antivírus
Criar senhas seguras
Evitar links desconhecidos
Não usar redes públicas inseguras
Fazer backup dos dados
