# 🔐 Mini Guia de Estudos: Fundamentos de Segurança Cibernética
## 📌 Sobre o Projeto

Este repositório apresenta a construção de um Caderno Temático utilizando o NotebookLM, com foco em Segurança Cibernética.

O projeto tem como objetivo demonstrar como a Inteligência Artificial pode ser utilizada como ferramenta de aprendizagem ativa, auxiliando na organização, análise e consolidação do conhecimento técnico.

🎯 Objetivos de Aprendizagem
Compreender os conceitos fundamentais da segurança da informação
Identificar os principais tipos de ameaças cibernéticas
Aplicar boas práticas de proteção no ambiente digital
Desenvolver pensamento crítico no uso de IA
Estruturar conhecimento técnico de forma clara e reutilizável

🧠 Miniguia de Estudo
🔐 O que é Segurança Cibernética?

Segurança cibernética é o conjunto de práticas, tecnologias e processos utilizados para proteger sistemas, redes e dados contra ataques digitais e acessos não autorizados.

⚠️ Principais Ameaças
Phishing: tentativa de enganar usuários para roubar informações sensíveis
Malware: softwares maliciosos que comprometem sistemas
Ransomware: bloqueio de dados mediante pagamento
Ataques de força bruta: tentativa automatizada de descobrir senhas
🛡️ Boas Práticas de Segurança
Utilizar senhas fortes e únicas
Ativar autenticação em dois fatores (2FA)
Evitar clicar em links suspeitos
Manter sistemas atualizados
Utilizar antivírus e firewall
📖 Glossário
Firewall: sistema que controla o tráfego de rede
Malware: software malicioso
Phishing: técnica de fraude digital
Criptografia: método de proteção de dados por codificação
Autenticação: processo de verificação de identidade
